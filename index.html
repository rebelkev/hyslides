HySlides helpQuick instructions for creating and presenting.

Create a presentation

Add blank slides or choose a layout from Templates. Changes autosave in this browser; Save now forces an immediate save.

Present and invite participants

Select Present to open Presenter View. Participants join using the QR code or six-digit access code and follow the current slide from their own device.

Use audience engagement

Add an Engagement element or engagement template, customize its choices, then go live. Results update on the slide as responses arrive.

What are the four views?

Editor View builds slides. Presentation View is projected to the room. Participant View appears on audience devices. Presenter View controls the live presentation.

Where are saved decks?

Open the HySlides menu and choose Open saved decks. Decks currently remain in this browser until user accounts and cloud storage are introduced.

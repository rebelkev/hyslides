# HySlides Documentation

This folder is the maintained reference for HySlides. Update it whenever product behavior, deployment, data storage, or known limitations change.

- [Product Guide](PRODUCT_GUIDE.md) — terminology, features, and instructions for creators, presenters, and participants.
- [Technical Handoff](TECHNICAL_HANDOFF.md) — architecture, data model, hosting, testing, retention, and developer orientation.
- [Roadmap](ROADMAP.md) — completed foundations, next milestones, and intentionally deferred work.
- [Release Notes](RELEASE_NOTES.md) — user-visible changes grouped by development milestone.

## Canonical View Names

Use these names consistently in UI copy, issues, documentation, and conversation:

1. **Editor View**
2. **Presentation View**
3. **Participant View**
4. **Presenter View** — Desktop and Mobile variants

Avoid “Audience View,” because it can ambiguously mean Presentation View or Participant View.
